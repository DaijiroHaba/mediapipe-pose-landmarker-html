MediaPipe Pose Landmarker

動画をサーバーへアップロードせず、ブラウザ内で4方向の姿勢ランドマークを解析します。

Static HTML + JavaScript

研究データ保護: 選択した動画はこのページ内で処理され、アプリ側サーバーへ送信されません。GitHub PagesやNetlifyなどの静的ホスティングで公開できます。

4方向対応: 前方、側方、後方、その他の動画を任意に選択できます。少なくとも1つ選択すれば解析できます。

長時間動画: sample fps は通常 2、長時間動画では 0.5〜1 を推奨します。ブラウザ性能、メモリ、動画サイズに処理時間が左右されます。

動画形式: ブラウザ標準の MediaRecorder はChrome/EdgeでWebM出力が最も安定しているため、ランドマーク付き動画は .webm で保存します。
